    🤖 LLM 模型

    默认空；只有显式选择备选 Provider 时才会 fallback，不再自动尝试其它 Provider。

    适配 Groq / Together / Azure OpenAI / vLLM / 自建等任何走 OpenAI 协议的服务。Base URL 必填，否则等同于普通 OpenAI provider。

    Embedding 与 LLM 各自独立配置，Provider 留空时不会跟随默认 LLM。Claude / DeepSeek / OpenRouter 没有 embedding 接口，未在此列出。

    ⏰ 调度

    开启后会暂停定时发现、候选池预计算和画像更新中的 LLM / embedding 调用；推荐列表不再自动补充，候选池为空时可能暂时没有推荐，建议初始化完成并已有内容后再开启。

    开启后，当所有扩展窗口断开时，后台 LLM / embedding 工作会在宽限期后自动暂停；重新打开浏览器后恢复，离线期间不会自动补新内容。

    此开关不会更新浏览器插件；插件商店安装由浏览器处理，zip / 开发者模式加载需要手动下载新版并重新加载。
